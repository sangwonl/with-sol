- 20260118 솔이와 정리
- 제목: Smart Drop Game
- HUD 와 게임 중 본론인 게임을 먼저 구현해보자
	- 수학, 영어, 한자 중에 수학 먼저 해보자
	  - 구성은 덧셈, 뺄셈, 곱셈, 나눗셈, Mix 가 있음
	  - 이 중에서 덧셈을 먼저 해보자
- HUD 씬
  - 처음 화면
    - SMART DROP 로고 있고
    - 수학 / 한자 / 영어 (우선 수학만 설정 가능)
    - 각 과목별로 Best Score 가 나옴
    - 끝내기 버튼도 있음
    - 만약 과목을 선택하면 게임 설정 화면으로 이동
  - 설정 화면
    - 한 문제당 몇초인지 설정이 있음
    - 문제는 일단 20개로 고정
    - 한 문제당 설정 가능 시간은 - 2초, 5초, 10초, 20초가 가능
      - 문제 난이도 설정이 있음
        - 쉬움: 한자리 + 한자리
        - 보통: 두자리 + 두자리 이하
        - 어려움: 세자리 + 세자리 이하 
- 게임 씬
  - 맨위에는 현재까지 맞은 개수(예를들면, 3/20), 점수(한문제당 5점, 전체 100점)
  - 가운데에는 문제가 보임
    - 문제가 위에서부터 떨어짐(한문제당 시간에 따라 속도가 결정된다)
    - 아래에 선이 있는데 이 선에 닿기 전에 정답을 맞춰야 함
  - 아래쪽에는 정답 선택 버튼이 있음
    - 5개가 기본으로 있음
    - 5개 중에 정답은 하나가 있고 어느 위치에 있을지는 모름
    - 정답을 누르면 현재 맞춘 개수와 점수가 올라가고 문제는 사라지고 다음 문제가 내려옴
    - 정답을 못 맞추면 문제가 그냥 사라지고 다음 문제가 내려옴
  - 효과음
    - 맞으면 딩동, 틀리면 띠용 (SKY VOCA 참고)
- 게임이 종료되면
  - 맞은 개수, 점수, 총 걸린 시간이 나타나고 멘트가 나옴
  - 90점 이상: 대단해요! / 70점 이상: 잘했어요 / 60점 이하: 우리 같이 더 노력해요
  - 확인 버튼을 누르면 처음 화면으로 돌아감

- 용어
  - 맞춘 개수 -> Solved (Question)
  - 점수 -> Score
  - 문제 자체 -> Question
  - 선택지 -> Mini Box
  - 총 풀이시간 -> Total Time
  - 멘트 -> Comment
